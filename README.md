# calculas-focused-calculater

A powerful calculus-focused graphing calculator with Taylor polynomial approximation, symbolic differentiation, numerical integration, limits, equation solving, ODE solving, and more. Features a Desmos-inspired UI with multiple function support, variable sliders, a virtual math keyboard, and OCR image-to-expression recognition. Supports up to 200 Taylor series terms and can be used to study the behavior of Taylor series approximations in real time.
